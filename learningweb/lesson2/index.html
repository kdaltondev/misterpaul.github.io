Super Simple Web Page

Here is another simple web page. What changes do you notice?

If you look in the

<head>
section, you'll see a link to a stylesheet. The stylesheet lets us define styles, that change how the page and its components look. If you change from the Elements tab to the Sources tab, you should see the file
index.css
listed. That is the stylesheet.

Now go back to the Elements tab and look at the index.html file again. Notice that some of the

<p>
tags now also contain
class=box
. You'll also see
div
tags that call the class
code
.

Go back to the index.css file (under Sourcs). Change the background color of box to red. Yuck. Find a better color. Play around with the font in the class

code
.

Unfortunately, to modify the HTML code, you need to be in the Elements tab, and to modify the stylesheet, you need to be in the Sources tab, so you may have to jumb back and forth a bit.

Challenge: Edit the stylesheet and create a new style called

.challenge
and set the background-color and text color (called just color). Then edit the HTML and set this paragraph to your
challenge
style. (note that you do not use the dot in the HTML)
