Super Simple Web Page

Here is another simple web page. What changes do you notice?

If you look in the <head> section, you'll see a link to a stylesheet. The stylesheet lets us define styles, that change how the page and its components look. If you change from the Elements tab to the Sources tab, you should see the file index.css listed. That is the stylesheet.

Now look at the index.html file again. (You can look at it under Elements again, or under the Sources tab.) Notice that some of the <p> tags now also contain class=box.
